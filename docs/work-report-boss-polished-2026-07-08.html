关键思考

这次工作的价值在于把风险前置：未来节点类型变多、外部源接入、LLM 自动抽取增多时，系统仍然有边界。

不是让 LLM 自由构图，而是让 LLM 在证据、结构和状态机约束下提出候选，再由治理机制决定是否进入生效图。
第一层

所有 LLM 产物先进入 proposed，不直接参与图算法和教学导出。

第二层

只有证据、结构佐证和复核结论同时成立，才允许自动批准为 approved。

第三层

边写入前检查端点类型，运行后由守卫扫描存量数据，形成前后两道保护。

第四层

审核日志和批次回滚让自动化结果可追责，发现污染可以按批次撤销。
